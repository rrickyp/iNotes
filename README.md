# iNotes
A web page for taking notes and store it in MongoDB
# To start the react app
1. Go to NoteService folder and run "node app.js"
2. ./bin/mongod --dbpath YourPath/NoteService/data
3. ./bin/mongo --dbpath YourPath/NoteService/generate_db.js
4. go to noteapp folder and run "npm start"
Please make sure that you have the icon for the user since, the App.js (in noteapp folder will crash if you do not have it)
